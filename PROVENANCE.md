## Principles

All source code, documentation, translations, tests, and examples added to Harbour Rust must be original, properly licensed, or otherwise lawful to redistribute.

## Allowed Inputs

- original implementation work;
- original documentation and translations;
- behavior observations derived from running tools and test fixtures;
- short factual references to public specifications or license-compatible material;
- small attributed excerpts when legally necessary and clearly justified.

## Disallowed Inputs

- copying substantial code from `harbour-core` or any other project;
- copying manuals, books, blog posts, or tutorials into repository documentation;
- importing proprietary test corpora without explicit permission;
- using trademarks in a way that implies affiliation or endorsement.

## Upstream Reference Policy

`harbour-core` may be used as:

- a behavior oracle;
- an architectural map;
- a source of test inspiration;
- a reference for compatibility research.

It must not be used as a source for large direct code translation or documentation copying.

## Translation Policy

- English is canonical.
- Portuguese is a maintained translation.
- Translations must preserve meaning, not mimic wording mechanically.
- Machine assistance is acceptable only with human review and originality checks.

By contributing, you represent that you have the right to submit the material under this repository's license and policies.
