# Migração da Documentação Técnica

Este diretório é o destino planejado para a documentação técnica em Português.

Estado atual do repositório:

- o projeto já possui notas técnicas ativas em `docs/*.md`;
- esses arquivos continuam válidos para o fluxo atual de contribuição;
- futuros guias técnicos voltados ao público devem migrar para esta estrutura espelhada em Inglês e Português.

Regra de migração:

- quando um documento técnico for revisado de forma substancial para uso público, mova-o ou reescreva-o na estrutura bilíngue em vez de ampliar o padrão legado de arquivo único.
