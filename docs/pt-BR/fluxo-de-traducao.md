# Fluxo de Tradução

## Fluxo Padrão

1. Atualize o documento fonte em Inglês.
2. Atualize o espelho em Português na mesma branch sempre que possível.
3. Verifique se estrutura, títulos, links e decisões continuam correspondentes.
4. Registre qualquer desencontro temporário explicitamente no pull request.

## Regras

- não trate saída de máquina como texto final;
- revise terminologia para manter consistência entre as camadas do compilador;
- preserve intenção, escopo e significado jurídico;
- mantenha as duas versões igualmente úteis para leitura.

## Checklist Recomendado para Pull Request

- arquivo em Inglês atualizado
- arquivo em Português atualizado
- links conferidos
- implicações legais/de proveniência revisadas
- qualquer tradução adiada documentada
