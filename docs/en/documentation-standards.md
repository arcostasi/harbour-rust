# Documentation Standards

## Goals

- keep public documentation clear, durable, and easy to review;
- ensure English and Portuguese stay structurally aligned;
- reduce legal and editorial risk.

## Canonical Language

English is canonical for public documentation. Portuguese is the maintained mirror.

## Structural Rules

- mirror headings and section order across language pairs;
- keep filenames stable and predictable;
- prefer short, explicit titles;
- separate policy, onboarding, and technical reference material.

## Writing Rules

- write in original language, do not paraphrase copyrighted text from other sources;
- prefer precise, low-hype language;
- identify known limits clearly;
- state assumptions when behavior is incomplete or provisional.

## Review Rules

- any public-doc change should check whether a mirrored file must also change;
- translation review should verify meaning, not word-for-word similarity;
- legal or attribution-sensitive changes should also be checked against `PROVENANCE.md`.
