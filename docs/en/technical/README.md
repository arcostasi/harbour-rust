# Technical Documentation Migration

This directory is the target home for English technical documentation.

Current repository state:

- the project already has active technical notes under `docs/*.md`;
- those files remain authoritative for current contributor workflows;
- future public-facing technical guides should migrate here in mirrored English and Portuguese pairs.

Migration rule:

- when a technical document is substantially revised for public use, move or rewrite it into the bilingual structure rather than extending the legacy single-file pattern.
