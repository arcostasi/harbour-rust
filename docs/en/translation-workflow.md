# Translation Workflow

## Default Flow

1. Update the English source document.
2. Update the Portuguese mirror in the same branch whenever possible.
3. Verify that structure, headings, links, and decisions still match.
4. Note any temporary mismatch explicitly in the pull request.

## Rules

- do not treat machine output as final text;
- review terminology for consistency across compiler layers;
- preserve intent, scope, and legal meaning;
- keep both versions equally usable for readers.

## Recommended Pull Request Checklist

- English file updated
- Portuguese file updated
- links checked
- legal/provenance implications reviewed
- any intentionally deferred translation documented
