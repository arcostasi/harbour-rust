# Release 0.4.0-alpha

## Escopo

Esta release consolida o recorte das Fases 10, 11 e 12:

- `harbour-rust-rdd` com DBF/RDD inicial para abertura, navegação, leitura e mutação persistente básica,
- CLI com `help`, `check`, `transpile --to c`, `run` e códigos de saída coerentes,
- harness de golden tests, comparador com runner Harbour externo, benchmark smoke e skeleton de fuzzing,
- workflows de qualidade e release para preflight, benchmark report e build release do CLI.

## Preflight local da release

```text
cargo fmt --all --check
cargo clippy --workspace --all-targets --all-features -- -D warnings
cargo test --workspace
cargo check --manifest-path fuzz/Cargo.toml
cargo run -p harbour-rust-tests --bin benchmark-suite -- --fixture examples/hello.prg --iterations 1
cargo build --release -p harbour-rust-cli
```

## Estado conhecido

- backend principal continua `codegen-c`, sem backend nativo ainda,
- `docs/harbour/` permanece checkout/documentação auxiliar fora do escopo da release,
- divergências abertas continuam documentadas em `COMPATIBILITY.md`,
- comparação automática com Harbour ainda depende de runner externo configurado localmente.

## Artefatos esperados
